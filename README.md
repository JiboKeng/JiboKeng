### 🚀 What I Do:
* **Multi-Container Architecture Integration:** Architecting isolated, microservice-driven networks using Docker and Docker Compose to decouple analytics interfaces, relational persistence databases, and experiment tracking nodes.
* **Production-Grade API Deployment:** Designing lightweight, production-ready REST API gateways using FastAPI to serialize and serve complex analytical model payloads securely to client networks.
* **Predictive Modeling & Risk Assessment:** Building and optimizing supervised machine learning classifiers designed for high-stakes precision environments, utilizing custom evaluation metrics tailored explicitly for sensitivity optimization (Recall/F1-Score).
* **MLOps Experiment Tracking:** Implementing robust experiment management frameworks utilizing MLflow to maintain auditable tracking of parameters, clinical metric thresholds, and serialized artifact registries.
* **Business Intelligence & Analytics:** Translating dense backend relational database structures into interactive, stakeholder-ready Power BI dashboards.

### 📈 Current Focus:
